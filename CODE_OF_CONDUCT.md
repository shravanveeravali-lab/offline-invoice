# Code of Conduct

## Our Pledge

We pledge to make participation in Offline Invoice Intelligence System respectful, inclusive, and focused on constructive technical collaboration.

## Expected Behavior

- Communicate respectfully.
- Give feedback that is specific and actionable.
- Credit teammates and contributors fairly.
- Respect privacy and avoid sharing sensitive invoice data.
- Keep discussions aligned with the project goals.
- Help maintain a welcoming environment for students, mentors, reviewers, and judges.

## Unacceptable Behavior

- Harassment, insults, or discriminatory language.
- Publishing private information without consent.
- Sharing real invoice data, tax identifiers, bank details, or personal information.
- Introducing hidden cloud calls or dependencies that violate offline requirements.
- Disruptive behavior during reviews, demos, or planning.

## Project Values

- Privacy by design
- Offline-first engineering
- Open-source collaboration
- Practical CPU optimization
- Clear documentation
- Respectful teamwork

## Enforcement

Maintainers may remove content, reject contributions, or restrict participation if behavior violates this code of conduct.

## Reporting

Report concerns privately to the project maintainers. Reports should include relevant context, links, screenshots, or reproduction steps when appropriate.
